## Problem Statement

Print the given pattern.

```text

E D C B A 
D C B A 
C B A 
B A 
A 

```

## Approach

- In the given pattern, we can see there are **n** rows and **n** columns.
- From the given pattern, we can deduce another number pattern as shown in below table
- In the number pattern, we can deduce that on each row, **n - rowIndex + 1** numbers of columns are printed. And value of the columns are traversing from **n - rowIndex** down to **0**
- On adding the above number with **'A'**, we can get our required pattern

```kt
fun pattern33(n: Int) {
    for(rowIndex in 1..n) {
        for(colIndex in (n - rowIndex) downTo 0) {
            print("${'A' + colIndex} ")
        }
        println()
    }
}

fun main() {    
    pattern33(5)
}

```
