# Pattern 15

## Problem Statement

Print the given pattern.

```text

    * 
   * * 
  *   * 
 *     * 
*       * 
 *     * 
  *   * 
   * * 
    *

```
## Approach

- In the given pattern, we can see there are **2n - 1** rows and **2n - 1** columns.
- We have to print some spaces before printing the **\*** characters.
- We have to print prefix spaces equal to, if **rowIndex <= n** then **n - rowIndex** else **rowIndex - n**.
- For columns, if **rowIndex <= n** then, we have to loop from **1** to **2 * rowIndex - 1** else, we have to loop from **1** to **2 * (n + n - rowIndex) - 1**, and print **\*** for edge index else print space.

```kt
fun pattern15(n: Int) {
    for(rowIndex in 1..(n + n - 1)) {
		val noOfCols = if(rowIndex <= n) 2 * rowIndex - 1 else 2 * (n + n - rowIndex) - 1
        val noOfSpaces = if(rowIndex <= n) n - rowIndex else rowIndex - n
        print(" ".repeat(noOfSpaces))
        for(colIndex in 1..noOfCols) {
            if(colIndex == 1 || colIndex == noOfCols)
                print("*")
            else
                print(" ")
        }
        println()
    }
}

fun main() {	
    pattern15(5)
}
```
