# Pattern 9

https://github.com/kunal-kushwaha/DSA-Bootcamp-Java/blob/main/assignments/09-patterns.md

## Problem Statement

Print the given pattern.

```text

*********
 *******
  *****
   ***
    *

```
## Approach

- In the given pattern, we can see there are **n** rows and **2n - 1** columns.
- We have to print some spaces before printing the **\*** characters.
- As we can see we have to print **rowIndex - 1** number of spaces and **2 * (n - rowIndex + 1) - 1** number of columns 
- For example: for **rowIndex = 2**, we can see we have to print **1** spaces and **7** columns. 
```kt
noOfSpaces = rowIndex - 1 = 2 - 1 = 1
noOfCols = 2 * (n - rowIndex + 1) - 1 = 2 * (5 - 2 + 1) - 1 = 7
```

```kt
fun pattern9(n: Int) {
    for(rowIndex in 1..n) {
		val noOfCols = 2 * (n - rowIndex + 1) - 1
        val noOfSpaces = rowIndex - 1
        print(" ".repeat(noOfSpaces))
        print("*".repeat(noOfCols))
        println()
    }
}

fun main() {	
    pattern9(5)
}
```
