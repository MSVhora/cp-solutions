# Pattern 8

## Problem Link

https://github.com/kunal-kushwaha/DSA-Bootcamp-Java/blob/main/assignments/09-patterns.md

## Problem Statement

Print the given patterns

```text

    *
   ***
  *****
 *******
*********

```
## Approach

- In the given pattern, we can see there are **n** rows and **2n - 1** columns.
- We have to print some spaces before printing the **\*** characters.
- As we can see we have to print **n - rowIndex** number of spaces and **2 * rowIndex - 1** number of columns 
- For example: for **rowIndex = 3**, we can see we have to print **2** spaces and **5** columns. 
```kt
noOfSpaces = n - rowIndex = 5 - 3 = 2
noOfCols = 2 * rowIndex - 1 = 2 * 3 - 1 = 5
```

```kt
fun pattern8(n: Int) {
    for(rowIndex in 1..n) {
		val noOfCols = 2 * rowIndex - 1
        val noOfSpaces = (n - rowIndex)
        print(" ".repeat(noOfSpaces))
        print("*".repeat(noOfCols))
        println()
    }
}

fun main() {	
    pattern8(5)
}
```
