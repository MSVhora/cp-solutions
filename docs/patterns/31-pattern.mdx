## Problem Statement

Print the given pattern.

```text

E 
D E 
C D E 
B C D E 
A B C D E 

```

## Approach

- In the given pattern, we can see there are **n** rows and **n** columns.
- From the given pattern, we can deduce another number pattern as shown in below table
- In the number pattern, we can deduce that on each row, **rowIndex** numbers of columns are printed. And value of the columns are traversing from **n - rowIndex + 1** to **n**
- On adding the above number with 'A' - 1, we can get our required pattern

```kt
fun pattern31(n: Int) {
    for(rowIndex in 1..n) {
        for(colIndex in (n - rowIndex + 1)..n) {
            print("${'A' + colIndex - 1} ")
        }
        println()
    }
}

fun main() {    
    pattern31(5)
}

```
