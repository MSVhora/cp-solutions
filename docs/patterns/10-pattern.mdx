# Pattern 10

https://github.com/kunal-kushwaha/DSA-Bootcamp-Java/blob/main/assignments/09-patterns.md

## Problem Statement

Print the given pattern.

```text

    * 
   * * 
  * * * 
 * * * * 
* * * * * 

```
## Approach

- In the given pattern, we can see there are **n** rows and **n** columns.
- We have to print some spaces before printing the **\*** characters.
- As we can see we have to print **n - rowIndex** number of spaces and **rowIndex** number of columns. 
- We also have to print an extra space between character, so while printing character we will print a space as well.

```kt
fun pattern10(n: Int) {
    for(rowIndex in 1..n) {
		val noOfCols = rowIndex
        val noOfSpaces = (n - rowIndex)
        print(" ".repeat(noOfSpaces))
        print("* ".repeat(noOfCols))
        println()
    }
}

fun main() {	
    pattern10(5)
}
```
