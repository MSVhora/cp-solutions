# Pattern 14

## Problem Link

https://github.com/kunal-kushwaha/DSA-Bootcamp-Java/blob/main/assignments/09-patterns.md

## Problem Statement

Print the given patterns

```text

********* 
 *     * 
  *   * 
   * * 
    * 

```
## Approach

- In the given pattern, we can see there are **n** rows and **2n - 1** columns.
- We have to print some spaces before printing the **\*** characters.
- As we can see we have to print **rowIndex - 1** number of spaces.
- For columns we have to loop from **1** to **2 * (n - rowIndex + 1) - 1**, and print **\*** for edge index else print space.
- Lastly for first row, we have to print all **\*** character for every column

```kt
fun pattern14(n: Int) {
    for(rowIndex in 1..n) {
		val noOfCols = 2 * (n - rowIndex + 1) -1
        val noOfSpaces = rowIndex - 1
        print(" ".repeat(noOfSpaces))
        
        for(colIndex in 1..noOfCols) {
            if(rowIndex == 1 || colIndex == 1 || colIndex == noOfCols)
                print("*")
            else
                print(" ")
        }
        println()
    }
}

fun main() {	
    pattern14(5)
}
```
