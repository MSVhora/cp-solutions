## Problem Statement

Print the given pattern.

```text

* * * * * 
 * * * * 
  * * * 
   * * 
    * 
    * 
   * * 
  * * * 
 * * * * 
* * * * * 

```

## Approach

- In the given pattern, we can see there are **2n** rows and **n** columns.
- We have to print some spaces before printing the **\*** characters.
- In the given pattern, we can see each row prints **n - rowIndex + 1** number of columns till **n** and after **n** it print **rowIndex - n** number of columns.
- From the table we can deduce that we need to print spaces equal to **n - noOfCols** for each row.
- We also have to print an extra space between character, so while printing character we will print a space as well.

```kt
fun pattern12(n: Int) {
    for(rowIndex in 1..(n+n)) {
        val noOfCols = if(rowIndex <= n) (n - rowIndex + 1) else (rowIndex - n)
        val noOfSpaces = (n - noOfCols)
        print(" ".repeat(noOfSpaces))
        print("* ".repeat(noOfCols))
        println()
    }
}

fun main() {    
    pattern12(5)
}
```
