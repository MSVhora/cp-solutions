# Pattern 19

## Problem Link

https://github.com/kunal-kushwaha/DSA-Bootcamp-Java/blob/main/assignments/09-patterns.md

## Problem Statement

Print the given pattern.

```text

*        *
**      **
***    ***
****  ****
**********
****  ****
***    ***
**      **
*        *

```
## Approach

- In the given pattern, we can see there are **2n - 1** rows and **2n** columns.
- For columns, if **rowIndex <= n** then
  - print **rowIndex** columns
  - print **n - rowIndex** spaces in between 
  - print **n - rowIndex** spaces in between
  - print **rowIndex** columns
- else
  - print **2n - rowIndex** columns
  - print **rowIndex - n** spaces in between 
  - print **rowIndex - n** spaces in between 
  - print **2n - rowIndex** columns
- Examples: 
  - For **rowIndex = 2**,
    ```kt
    noOfCols = rowIndex = 2
    noOfSpaces = n - rowIndex = 5 - 2 = 3
    ```
  - For **rowIndex = 8**,
    ```kt
    noOfCols = 2n - rowIndex = 10 - 8 = 2
    noOfSpaces = rowIndex - n = 8 - 5 = 3
    ```

```kt
fun pattern19(n: Int) {
    for(rowIndex in 1..(n + n - 1)) {
		val noOfCols = if(rowIndex <= n) rowIndex else n + n - rowIndex
        val noOfSpaces = if(rowIndex <= n) (n - rowIndex) else rowIndex - n
        print("*".repeat(noOfCols))
        print(" ".repeat(noOfSpaces * 2))
        print("*".repeat(noOfCols))
        println()
    }
}

fun main() {	
    pattern19(5)
}
```
