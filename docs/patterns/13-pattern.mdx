## Problem Statement

Print the given pattern.

```text

    * 
   * * 
  *   * 
 *     * 
********* 

```

## Approach

- In the given pattern, we can see there are **n** rows and **2n - 1** columns.
- We have to print some spaces before printing the **\*** characters.
- As we can see we have to print **n - rowIndex** number of spaces.
- For columns we have to loop from **1** to **2 * rowIndex - 1**, and print **\*** for edge index else print space.
- Lastly for last row, we have to print all **\*** character for every column

|   5   |   *   |   *   |   *   |   *   |   *   |   *   |   *   |   *   |   *   |

```kt
fun pattern13(n: Int) {
    for(rowIndex in 1..n) {
        val noOfCols = 2 * rowIndex -1
        val noOfSpaces = n - rowIndex
        print(" ".repeat(noOfSpaces))
        
        for(colIndex in 1..noOfCols) {
            if(rowIndex == n || colIndex == 1 || colIndex == noOfCols)
                print("*")
            else
                print(" ")
        }
        println()
    }
}

fun main() {    
    pattern13(5)
}
```
