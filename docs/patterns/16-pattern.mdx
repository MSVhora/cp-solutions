## Problem Statement

Print the given pattern.

```text

        1   
      1   1   
    1   2   1   
  1   3   3   1   
1   4   6   4   1   

```

## Approach

- In the given pattern, we can see there are **n** rows and **n** columns.
- We have to print some spaces before printing the **\*** characters.
- As we can see we have to print **n - rowIndex** number of spaces.
- For Pascal Triangle values, [check here](https://www.geeksforgeeks.org/pascal-triangle)

```kt
fun pattern16(n: Int) {
    for(rowIndex in 1..n) {
        var value = 1
        val noOfSpaces = n - rowIndex
        print(" ".repeat(noOfSpaces * 2))
        for(col in 1..n) {
            if(value != 0)
                print("$value")
                print(" ".repeat(3))
            value = value * (rowIndex - col) / col
        }
        println()
    }
}

fun main() {    
    pattern16(5)
}
```
