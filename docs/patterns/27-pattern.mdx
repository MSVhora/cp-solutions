# Pattern 27

## Problem Statement

Print the given pattern.

```text

1  2  3  4  5   26 27 28 29 30 
   6  7  8  9   22 23 24 25 
      10 11 12  19 20 21 
         13 14  17 18 
            15  16 

```
## Approach

- In the given pattern, we can see there are **n** rows and **2n** columns.
- We have to print some spaces before printing the **\*** characters.
- As we can see we have to print **rowIndex - 1** number of spaces and **2 * (n - rowIndex + 1)** number of columns 
- **padEnd** method is used to add padding to number, so that it will display properly.

```kt
fun pattern27(n: Int) {
    var value = 1
    for(rowIndex in 1..n) {
		val noOfCols = n - rowIndex + 1
        val noOfSpaces = rowIndex - 1
        print(" ".repeat(noOfSpaces * 3))
        var curValue = value
        for(col in 1..noOfCols) {
            print("${curValue++}".padEnd(3))
        }
        print(" ")
        curValue = value
        for(col in 1..noOfCols) {
            val calcValue = curValue++ + noOfCols * noOfCols
            print("$calcValue".padEnd(3))
        }
        value = curValue
        println()
    }
}

fun main() {	
    pattern27(5)
}
```
