# Pattern 32

## Problem Statement

Print the given pattern.

```text

a 
B c 
D e F 
g H i J 
k L m N o 

```
## Approach

- In the given pattern, we can see there are **n** rows and **n** columns.
- In the given pattern, each row has **rowIndex** number of columns.
- From the pattern, we can deduce, that alphabets are printed from **a** in ascending order and in alternate small and caps format.
- So for printing character, we created variable **index**, if **index** is even then print small alphabet at **index** else print capital alphabet at **index**.

fun pattern32(n: Int) {
    var index = 0
    for(rowIndex in 1..n) {
        for(colIndex in 1..rowIndex) {
            val charValue = if(index % 2 == 0) 'a' + index else 'A' + index
            index++
            print("$charValue ")
        }
        println()
    }
}

fun main() {	
    pattern32(5)
}
