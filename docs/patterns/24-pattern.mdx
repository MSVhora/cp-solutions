## Problem Statement

Print the given pattern.

```text

*        *
**      **
* *    * *
*  *  *  *
*   **   *
*   **   *
*  *  *  *
* *    * *
**      **
*        *
```

## Approach

- In the given pattern, we can see there are **2n** rows and **2n** columns.
- For columns, if **rowIndex <= n** then
  - Traverse from **1** to **rowIndex** and print * on only edge indices.
  - print **n - rowIndex** spaces in between
  - print **n - rowIndex** spaces in between
  - Traverse from **1** to **rowIndex** and print * on only edge indices.
- else
  - Traverse from **1** to **2n - rowIndex + 1** and print * on only edge indices.
  - print **rowIndex - n - 1** spaces in between
  - print **rowIndex - n - 1** spaces in between
  - Traverse from **1** to **2n - rowIndex + 1** and print * on only edge indices.
- Examples:
  - For **rowIndex = 2**,
  
    ```kt
    noOfCols = rowIndex = 2
    noOfSpaces = n - rowIndex = 5 - 2 = 3
    ```

  - For **rowIndex = 8**,
  
    ```kt
    noOfCols = 2n - rowIndex + 1 = 10 - 8 + 1 = 3
    noOfSpaces = rowIndex - n - 1 = 8 - 5 - 1 = 2
    ```

```kt
fun pattern24(n: Int) {
    for(rowIndex in 1..(n + n)) {
        val noOfCols = if(rowIndex <= n) rowIndex else n + n - rowIndex + 1
        val noOfSpaces = if(rowIndex <= n) (n - rowIndex) else rowIndex - n - 1
        
        // Print columns
        for(colIndex in 1..noOfCols) {
            if(colIndex == 1 || colIndex == noOfCols)
                print("*")
            else
                print(" ")
        }

        // Print Spaces
        print(" ".repeat(noOfSpaces * 2))
        
        // Print columns
        for(colIndex in 1..noOfCols) {
            if(colIndex == 1 || colIndex == noOfCols)
                print("*")
            else
                print(" ")
        }

        println()
    }
}

fun main() {    
    pattern24(5)
}
```
