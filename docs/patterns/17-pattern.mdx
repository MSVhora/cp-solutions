# Pattern 17

https://github.com/kunal-kushwaha/DSA-Bootcamp-Java/blob/main/assignments/09-patterns.md

## Problem Statement

Print the given pattern.

```text

    1
   212
  32123
 4321234
543212345
 4321234
  32123
   212
    1

```
## Approach

- In the given pattern, we can see there are **2n - 1** rows and **2n - 1** columns.
- We have to print some spaces before printing the **\*** characters.
- We have to print prefix spaces equal to, if **rowIndex <= n** then **n - rowIndex** else **rowIndex - n**.
- If **rowIndex <= n** then
  - Print columns from **rowIndex** to **1** and from **2** to **rowIndex**.
- else
  - Print columns from **2n - rowIndex** to **1** and from **2** to **2n - rowIndex**.

```kt
fun pattern17(n: Int) {
    for(rowIndex in 1..(n + n - 1)) {
        val noOfCols = if(rowIndex <= n) rowIndex else n + n - rowIndex
        val noOfSpaces = if(rowIndex <= n) n - rowIndex else rowIndex - n
        print(" ".repeat(noOfSpaces))
        for(col in noOfCols downTo 1) {       
            print("$col")
        }
        
        for(col in 2..noOfCols) {       
            print("$col")
        }
        println()
    }
}

fun main() {	
    pattern17(5)
}
```
